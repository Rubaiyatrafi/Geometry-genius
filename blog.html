Question#1 : What is DOM and What is the purpose of DOM?

Answer : The Document Object Model (DOM) is a programming interface for web documents. The DOM represents the document as nodes and objects.

Question#2 : How will you select HTML elements using DOM? Name the DOM methods.

Answer : There are five ways to get HTML elements using DOM. One of them is getElementById()

Question#3 : What is event bubble?

Answer : Event Bubbling is a type of DOM event propagation. In here the event triggers lowest target element then successively triggers highest.
